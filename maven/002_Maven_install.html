                      # Java

                      检查下安装了否：

                      java -version

D:\aaron\workspace\project\xue\demo\demo>java -version
java version "18.0.1.1" 2022-04-22
Java(TM) SE Runtime Environment (build 18.0.1.1+2-6)
Java HotSpot(TM) 64-Bit Server VM (build 18.0.1.1+2-6, mixed mode, sharing)

                      如若没有安装， 安装 Java 软件开发套件（SDK）：Oracle JDK open in new window， 当然还有 Open JDK 等。

                      # 环境设置

                      设置 JAVA_HOME 环境变量，并指向你机器上的 Java 安装目录。例如：

                      操作系统例子
                      WindowC:\Program Files\Java\jdk-XXX 具体看你的安装版本
                      Linuxexport JAVA_HOME=/usr/local/java-current
                      MAC~/.bash_profile 加入 export JAVA_HOME=`/usr/libexec/java_home -v 18`

                      注意 linux 用soft link, MAC 用一个表达式，也就是一个环境可以有多个jdk 版本。

                      添加到系统目录

                      操作系统例子
                      WindowPath=%JAVA_HOME%\bin;<现有的其他路径> 注意是bin
                      Linuxexport PATH=PATH:PATH:PATH:JAVA_HOME/bin/
                      MAC~/.bash_profile 加入 export PATH=JAVAHOME/bin:JAVA_HOME/bin:JAVAH​OME/bin:PATH

                      Mac 加完后需要重新 source ~/.bash_profile PATH 如何设置也可以参考 oracle 官方文档open in new window。

                      # Maven

                      Apache maven downloadopen in new window 官方下载最新的包。

                      Maven 下载下来只是一个压缩包(zip),安装只是个解压的过程，所以可以解压到任何目录下。

                      操作系统位置 (根据你的安装位置而定)
                      WindowsE:\Maven\apache-maven-X.X.X
                      Linux/usr/local/apache-maven-X.X.X
                      Mac/usr/local/apache-maven-X.X.X

                      # Window

                      右键 "计算机"，选择 "属性"，之后点击 "高级系统设置"，点击"环境变量"，来设置环境变量，有以下系统变量需要配置：

                      1. 新建系统变量 MAVEN_HOME，变量值：E:\Maven\apache-maven-X.X.X
                      2. 编辑系统变量 Path，添加变量值：;%MAVEN_HOME%\bin

                      mvn --version 检查版本信息：

                      
D:\workspace\project\xue\>mvn --version
Apache Maven 3.8.5 (3599d3414f046de2324203b78ddcf9b5e4388aa0)
Maven home: D:\aaron\software\apache-maven-3.8.5
Java version: 18.0.1.1, vendor: Oracle Corporation, runtime: C:\Program Files\Java\jdk-18.0.1.1
Default locale: zh_CN, platform encoding: UTF-8
OS name: "windows 10", version: "10.0", arch: "amd64", family: "windows"

                      # Linux

                      下载解压：

                      # wget http://mirrors.hust.edu.cn/apache/maven/maven-3/3.3.9/binaries/apache-maven-3.3.9-bin.tar.gz
# tar -xvf  apache-maven-3.3.9-bin.tar.gz
# sudo mv -f apache-maven-3.3.9 /usr/local/

                      编辑 /etc/profile 文件 sudo vim /etc/profile，在文件末尾添加如下代码：

                      export MAVEN_HOME=/usr/local/apache-maven-3.3.9
export PATH=${PATH}:${MAVEN_HOME}/bin

                      保存文件，并运行如下命令使环境变量生效：

                      # source /etc/profile

                      在控制台输入如下命令，如果能看到 Maven 相关版本信息，则说明 Maven 已经安装成功：

                      # mvn -v

                      # Mac

                      下载解压：

                      $curl -O http://mirrors.hust.edu.cn/apache/maven/maven-3/3.3.9/binaries/apache-maven-3.3.9-bin.tar.gz
$tar -xvf  apache-maven-3.3.9-bin.tar.gz
$sudo mv -f apache-maven-3.3.9 /usr/local/

                      编辑 /etc/profile 文件 sudo vim /etc/profile，在文件末尾添加如下代码：

                      export MAVEN_HOME=/usr/local/apache-maven-3.3.9
export PATH=${PATH}:${MAVEN_HOME}/bin

                      保存文件，并运行如下命令使环境变量生效：

                      
$source /etc/profile

                      在控制台输入如下命令，如果能看到 Maven 相关版本信息，则说明 Maven 已经安装成功：

                      
$mvn -v
